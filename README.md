# CyberSec Password Utility 🔐

A modern, cybersecurity-themed Password Strength Checker and Secure Password Generator built as a single-page web application. This project features a striking dark cyberpunk aesthetic with glassmorphism, glowing neon accents, and subtle micro-animations.

It is perfect for a final-year cybersecurity mini-project or as a standout piece in a GitHub portfolio.

## 🚀 Features

- **Real-Time Strength Analysis:** Instantly calculates password strength as you type based on length, uppercase, lowercase, numbers, and symbols.
- **Dynamic Strength Meter:** Visual progress bar and score indicators (Weak, Medium, Strong).
- **Actionable Suggestions:** Provides a live checklist of what to improve in your password.
- **Advanced Password Generator:** 
  - Customizable length (8-32 characters).
  - Toggles for Uppercase, Lowercase, Numbers, and Symbols.
- **Cyberpunk UI:** Dark theme, neon colors, glassmorphism panels, and an animated background grid.
- **Micro-Interactions:** Custom styled sliders, checkboxes, glitch text animations, and hover effects.
- **One-Click Copy:** Easily copy your generated password to the clipboard with a toast notification.
- **Visibility Toggle:** Easily show or hide the entered password.
- **No Dependencies:** Built with 100% Vanilla HTML, CSS, and JavaScript.

## 📂 Project Structure

```text
cyber_password_tool/
├── index.html   # Main HTML5 structure and layout
├── style.css    # Cyberpunk styling, glassmorphism, and animations
```

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom variables, flexbox/grid, animations, transitions, and glassmorphism styling.
- **JavaScript (ES6):** DOM manipulation, password generation algorithms, and real-time validation logic.

## 💻 How to Run

Since this project has no external dependencies or backend requirements, you can run it directly in your browser:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/cyber-password-utility.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cyber-password-utility
   ```
3. Open `index.html` in your favorite modern web browser.
